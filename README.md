# stm

This files were used to shuffle the antimicrobial peptide sequences in order to generate the negative dataset used on sense the moment system.

Usage:

1. Clone this repository;
2. Edit the main.pl file, adding the fasta file with the sequences;
3. run 
    #perl main.pl

The output consists of five csv files, containing the shuffled averages by 1, 10, 100, 1000 and 10000 folds in 100 independent simulations.
